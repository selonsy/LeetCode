文件的命名规范：

目录：各语言的子目录 leetcode。
格式：leetcode_题号_英文名_中文名.后缀
注意：
1、英文名称之间用_连接。
2、特殊符号如（）则去掉，如：String to Integer (atoi)  改为：String_to_Integer_atoi
3、题号保留4位整数，不够的补0
举例：
题号：1
英文名：Two Sum
中文名：两数之和

结果：
leetcode_0001_Two_Sum_两数之和.java  //java版本
leetcode_0001_Two_Sum_两数之和.cpp   //c++版本
leetcode_0001_Two_Sum_两数之和.py    //python版本
